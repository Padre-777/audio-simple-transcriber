# Audio Simple Transcriber (Whisper) 🎙️

[![Open In Colab](https://google.com)](https://google.com)

Простой инструмент для автоматической расшифровки аудио в текст прямо в браузере. Использует модель **OpenAI Whisper (medium)** для обеспечения высокой точности.

## 🚀 Как пользоваться
1. Нажмите на кнопку **Open In Colab** выше.
2. Запустите ячейку (нажмите на иконку Play или `Ctrl+Enter`).
3. Загрузите ваш аудиофайл (поддерживаются `.aac`, `.mp3`, `.wav`).
4. Дождитесь окончания расшифровки. Результат будет сохранен в файл `transcription.txt`.

## 🛠 Особенности
*   **Модель:** Whisper `medium` (оптимальный баланс скорости и качества).
*   **Язык:** Автоматически настроен на русский.
*   **Библиотеки:** Использует `openai-whisper` и `ffmpeg`.

## 📋 Требования
Для быстрой работы в Colab рекомендуется включить GPU (**Среда выполнения** -> **Сменить тип среды выполнения** -> **T4 GPU**).
